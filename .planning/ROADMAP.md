# Roadmap: Social Casino Landing Page

## Overview

Building a modular, high-performance social casino landing page for the US market. The journey is split into scaffolding the design system and content, building the landing page sections, implementing a robust multi-step registration flow, and finally integrating with Google Sheets for data collection.

## Phases

- [x] **Phase 1: Foundation & Scaffolding** - Setup design tokens, content config, and layout skeleton.
- [x] **Phase 2: Landing Page Sections** - Build Hero, Social Proof, Games, and Footer.
- [x] **Phase 3: Multi-Step Form Logic** - Implement the 3-step registration flow with validation.
- [x] **Phase 4: Integrations** - Setup Google Sheets Apps Script and form submission handling.
- [x] **Phase 5: V1 Refinement (SpinQuest)** - Polish and verify visual style for v1.
- [x] **Phase 6: V2 Adaptation (Luck Is a Journey)** - Implement and verify visual style for v2.
- [x] **Phase 7: Optimization & QA** - Final performance tuning and compliance checks.
